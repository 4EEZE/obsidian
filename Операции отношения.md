В [[Java]] операции отношения работают со значениями переменных.

| Операция | Описание         |
| -------- | ---------------- |
| ==       | Равно            |
| !=       | Не равно         |
| >        | Больше           |
| <        | Меньше           |
| >=       | Больше или равно |
| <=       | Меньше или равно |
Результатом данных операций является значение типа [[boolean]].

`==` и `!=` можно применять к любым объектам в Java.
Однако `>=` и `<=` можно применять только к типам.
